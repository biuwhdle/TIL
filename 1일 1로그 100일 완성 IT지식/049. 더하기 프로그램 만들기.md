- 수 합산하기 파이썬 프로그램
  - 컴퓨터에 설치한 파이썬 프롬프트에 복사, 붙여넣기 하면 input 함수 호출이 즉시 해석 -> 작동하지 않음
  - addup.py 같은 별도의 파일로 저장, 해당 경로를 지정한 후 파일을 파이썬으로 실행

```
sum = 0
num = input()
while num != '0':
  sum = sum + int(num)
  num = input()
print("The sum is ", sum)
```
